## 1. 🚀 Features

This repository serves as a lightweight, serverless web application hosting interactive web tools and mini-games. It combines FastAPI's low-latency backend routing with Jinja2 frontend templating.

1. **FastAPI ASGI Engine:** Leverages FastAPI as an asynchronous web framework to serve routes, static files, and templates with low latency.
2. **Jinja2 Server-Side Templating:** Renders dynamic HTML templates seamlessly, providing modular web pages such as home, about, and custom error handling.
3. **Interactive Mini-Games:** Houses lightweight, client-side browser games written in HTML5, CSS3, and JavaScript.
4. **Custom 404 Error Handling:** Configured with a dedicated fallback error page to gracefully handle missing routes or non-existent URLs.
5. **SEO & Crawl Optimization:** Pre-configured with clean route mappings, `robots.txt`, and an automated `sitemap.xml` for structured search engine indexing.
6. **Serverless Deployment:** Engineered with `vercel.json` and modular FastAPI handlers for instant, global deployment on Vercel's edge network.

<hr>

## 2. 🛠️ Tech Stack

The core technologies powering the application backend, dynamic rendering engine, and deployment pipeline include:

* **Backend Framework: [FastAPI](https://fastapi.tiangolo.com) (v0.139.2)** - Asynchronous framework for low-latency request handling.
* **Templating Engine: [Jinja2](https://jinja.palletsprojects.com) (v3.1.6)** - Renders dynamic server-side HTML pages and handles route template layouts.
* **Frontend: HTML5, CSS3, Vanilla JavaScript** - Delivers responsive UI styling, asset structuring, and client-side game logic.
* **Deployment: [Vercel](https://vercel.com)** - Serverless hosting and edge-optimized global delivery.

## 3. 🎮 List of Projects

This repository hosts a curated collection of interactive, lightweight mini-games served directly through Jinja2 templates:

1. **[Tennis Game](https://webprojects.anikethchavare.com/games/tennis):** A responsive, arcade-style 2D tennis game rendered on an HTML5 canvas. Features interactive paddle controls, real-time ball physics, collision detection, and scorekeeping.
2. **[Rock Paper Scissors](https://webprojects.anikethchavare.com/games/rps):** A classic hand-game implementation where players compete against a randomized bot logic. Features real-time score tracking, dynamic round results, and styled UI choices.
